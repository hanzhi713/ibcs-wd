Code changes can be made independently.

Benefits of inheritance:
Reusability -- facility to use public methods of base class without rewriting the same
Extensibility -- extending the base class logic as per business logic of the derived class
Data hiding -- base class can decide to keep some data private so that it cannot be altered by the derived class
Overriding--With inheritance, we will be able to override the methods of the base class so that meaningful implementation of the base class method can be designed in the derived class.

Benefits of polymorphism:
Run-time polymorphism (over writing)& compile time polymorphism(over loading)
- Same interface could be used for creating methods with different implementations
- Reduces the volume of work in terms of distinguishing and handling various objects
- Supports building extensible systems
- Complete implementation can be replaced by using same method signatures

Station:
public class Station {
    private int lineNumber;
    private int zone;
    private String name;
    public Station(String name,int lineNumber,int zone){
        this.name=name;
        this.lineNumber=lineNumber;
        this.zone=zone;
    }
    public int getZone(){
        return this.zone;
    }
    public int getLineNumber(){
        return this.lineNumber;
    }
    public String getName(){
        return this.name;
    }
}
 
Tree:
public class Tree {
    Station station;
    Tree left;
    Tree right;
 
    public Tree(Station station, Tree left, Tree right) {
        this.station = station;
        this.left = left;
        this.right = right;
    }
 
    public Tree(Station station) {
        this.station = station;
        this.right = null;
        this.left = null;
    }
 
    public Tree(Station station, Tree left, String a) {
        this.station = station;
        this.left = left;
        this.right = null;
    }
 
    public Tree(Station station, String a, Tree right) {
        this.station = station;
        this.right = right;
        this.left = null;
    }
 
    public static int getZone(Tree rootNode, String stationName) {
        Tree treeNode = Tree.search(rootNode, stationName);
        return treeNode.station.getZone();
    }
 
    public static int getLineNumber(Tree rootNode, String stationName) {
        Tree treeNode = Tree.search(rootNode, stationName);
        return treeNode.station.getLineNumber();
    }
 
    public static Tree search(Tree treeNode, String stationName) {
        if (stationName.equals(treeNode.station.getName())) {
            return treeNode;
        } else if (stationName.compareTo(treeNode.station.getName()) < 0) {
            return Tree.search(treeNode.left, stationName);
        } else {
            return Tree.search(treeNode.right, stationName);
        }
    }
 
}
