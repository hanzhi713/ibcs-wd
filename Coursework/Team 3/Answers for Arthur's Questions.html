1) State the three basic functions of an autonomous car. [3]
  • knowing the car’s exact location
  • being able to perceive the car’s immediate environment
  • making the correct driving decisions to get from A to B.

2) Define and outline the role of sensor fusion. [2]
  • Sensor Fusion is the set of various sensors, used to
  • provide the data for constructing a detailed picture of the immediate environment

3) Explain the mechanism of GPS to locate the user’s position. [5]
  • User connects to at least 3 satellites
  • User sends signal to each satellite
  • User calculates the time for the signal to travel,
  • thereby determining the distance between the user and the satellite
  • Knowing the distance between the user and three satellites, a spatial coordinate could be obtained by finding the point of intersection between three spheres.
